## 📕 このGitHubリポジトリについて

書籍「AIエディタCursor完全ガイド」 について下記のサポートを行うリポジトリです。

- 書籍に掲載されているハンズオン用コード、生成されたサンプルコード、プロンプト、スクリーンショット画像の公開。
- 原稿を書き上げた後の Cursor のアップデートによる変更点の補足説明。
- **書籍を読んでいて、画面と説明が一致しない、説明と同じ手順を行うことができない、といったケースでは、このリポジトリの該当章をご覧ください。**
